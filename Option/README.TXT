Attempt to predict future prices on the futures MICEX index use Decision Tree
==============================================================================

The calculations are performed at the jupiter notebook using Python 3.7
==============================================================================

Necessary package:
1) Pandas for work with data
2) numpy for advanced use of numbers and formulas
3) matplotlib for graphics
4) seaborn for more graphics
5) sklearn for Decision Tree

The calculations will use the data obtained from the QUIK program using the lua script
Lua script has been added to add to this directory
